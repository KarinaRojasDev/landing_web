Proyectos Destacados

LiterAlura

Aplicación en Java con Spring Boot que permite buscar libros usando la API de Gutendex, registrar información de libros y autores en PostgreSQL, y consultar los datos por distintos criterios como idioma, año de nacimiento de los autores y descargas.

Funcionalidades principales:

  • Buscar libros en la API de Gutendex y registrarlos en la base de datos
  • Listar libros y autores registrados
  • Filtrar autores vivos en un año específico y libros por idioma
  • Mostrar información de descargas y detalles de cada libro

Tecnologías: Java 17+, Spring Boot 3, Spring Data JPA/Hibernate, PostgreSQL, Gutendex API, Maven

NeoFichaje

NeoFichaje es una aplicación Android diseñada para la gestión de fichajes, vacaciones y documentos en entornos laborales. Permite a los empleados registrar sus jornadas, solicitar permisos y consultar documentos, mientras que los administradores supervisan y gestionan su equipo de manera eficiente desde cualquier lugar.

Tecnologías principales:

  • Kotlin, XML, Android Studio
  • Firebase Authentication, Firestore, Cloud Storage
  • Google Sign-In, MaterialCalendarView

Funcionalidades clave:

  • Fichaje con geolocalización
  • Registro y login seguro con correo y Google
  • Gestión de vacaciones y permisos
  • Consulta y descarga de documentos (contratos, nóminas)
  • Panel de administrador para control de empleados y aprobación de solicitudes
  • Historial de actividad por usuario
